## 文字の種類
このライブラリで変換される文字の種類は、主に次の通りです。  
ラテン1補助 (00A0-00FF) は対象外です。

| コード | ブロック |
-|-
| 0020-007F | ASCII 文字 |
| 3040-309F | ひらがな |
| 30A0-30FF | カタカナ |
| FF00-FF5F | 全角 ASCII 文字 |
| FF60-FF9F | 半角カタカナ |

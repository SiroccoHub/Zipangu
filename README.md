## Setup
Zipangu は NuGet Gallery に登録されています。  
Zipangu を利用するには、Visual Studio の [NuGet パッケージの管理] でインストールするか、  
あるいは [パッケージ マネージャー コンソール] で次のコマンドを実行します。

```
Install-Package Zipangu
```
